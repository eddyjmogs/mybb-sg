[size=xx-large][b]Sistema Elemental[/b][/size][/align]

El sistema elemental representa uno de los pilares del desarrollo ninja, estructurando cómo se manifiestan las habilidades, el estilo de combate y el progreso de un personaje. Cada ninja puede dominar uno o varios elementos, permitiendo no solo el uso de técnicas naturales, sino también combinaciones más avanzadas que dan lugar a elementos únicos. Estas combinaciones pueden ser heredadas por linaje o desarrolladas mediante entrenamiento. Esta guía cubre desde los fundamentos elementales hasta las reglas de combinación, choque, ventajas, y el rol especial de las energías Yin y Yang. También detalla cómo ciertos clanes acceden a elementos avanzados mediante ventajas genéticas.

[hr]

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Slots Elementales [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]
Los elementos se adquieren y utilizan a través de un sistema de slots. Cada ninja tiene una cantidad limitada de slots elementales disponibles, y cada elemento básico o combinación avanzada ocupa una cantidad específica de estos espacios.

Los cinco elementos básicos ocupan un solo slot cada uno. En cambio, las combinaciones elementales como los Kekkei Genkai o Kekkei Tōta requieren uno o dos slots respectivamente, ya que representan fusiones complejas. [color=#FF4136]El límite máximo de slots que puede tener un personaje es de cinco[/color].

[hr]

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Elementos Básicos [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]

Existen cinco elementos básicos que forman la base del sistema elemental: [color=#FF4136]Katon (Fuego)[/color], [color=#1E92F7]Suiton (Agua)[/color], [color=#FF851B]Doton (Tierra)[/color], [color=#4CEA5E]Fuuton (Viento) [/color]y [color=#FFDC00]Raiton (Rayo)[/color]. Cada uno de estos elementos tiene una afinidad específica, ventajas y debilidades en relación con los otros. Estas relaciones definen los enfrentamientos elementales en los combates, especialmente al momento de chocar jutsus de elementos distintos.

Los usuarios pueden aprender uno o varios de los elementos básicos, y más adelante combinarlos para desbloquear elementos avanzados. El dominio de los elementos se desarrolla de forma progresiva y aleatoria. Para desbloquear un nuevo elemento, el usuario debe gastar [color=#FF4136]un slot elemental[/color] y haber adquirido previamente el [color=#FF4136]Árbol de Habilidad[/color] donde se ubicará dicho elemento. Una vez hecho esto, se realiza una tirada aleatoria que determinará qué elemento adoptará ese Árbol de Habilidad. Para clanes con límites en sus elementos iniciales al momento de crear la ficha (como Yuki con Suiton y Fūton), la tirada aleatoria se hace entre los elementos dispuestos dentro del límite. Los clanes Mono-elementales (como Hoshigaki), evidentemente no hacen tirada.

Cada uno de los cinco elementos básicos cuenta con su propio árbol elemental predeterminado dentro del foro. Sin embargo, esto no limita al usuario a ceñirse únicamente a ellos: puede crear árboles personalizados, siempre y cuando se respete la afinidad elemental correspondiente y el sistema de creaciones únicas. Asimismo, es posible que un personaje posea más de un Árbol de Habilidad asociado al mismo elemento, tanto en el caso de los Elementos Básicos como en el de los Elementos Avanzados.

[hr]

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Elementos Avanzados [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]

Cuando un personaje domina dos o más elementos básicos, puede combinarlos para crear un nuevo elemento más avanzado. Estas fusiones se dividen en dos categorías: [b]Kekkei Genkai[/b] (combinación de dos elementos) y [b]Kekkei Tōta[/b] (combinación de tres). Para poder acceder a un elemento avanzado, todos los usuarios que hayan alcanzado el nivel 5 pueden realizar una tirada especial con una probabilidad de éxito de [b]1 entre 20[/b]. Si el resultado es favorable, obtienen el derecho a crear un nuevo elemento avanzado mediante el sistema de creaciones. Además, como beneficio adicional, todas las técnicas que formen parte del [b]primer Árbol de Habilidades[/b] creado para ese nuevo elemento serán completamente gratuitas.

[b][color=#FF4136]Kekkei Genkai (Fusión de dos elementos)[/color][/b]
Los Kekkei Genkai elementales son combinaciones de dos elementos básicos. Para utilizarlos, el usuario debe poseer ambos elementos base y además invertir [color=#FF4136]un slot elemental[/color] para habilitar el uso del elemento combinado. Esta fusión permite la creación de técnicas únicas y especializadas que no se podrían lograr con los elementos por separado.

Algunos ejemplos clásicos de Kekkei Genkai son:

[list]
[*][color=#9DF9FF]Hyoton (Hielo):[/color] combinación de Suiton y Fuuton.
[*][color=#FF5F54]Yoton (Lava):[/color] combinación de Doton y Katon.
[/list]

Cada uno de estos elementos fusionados hereda fortalezas y debilidades de sus elementos base, aunque funcionan con reglas especiales en cuanto a ventajas y choques.

[b][color=#FF4136]Kekkei Tōta (Fusión de tres elementos)[/color][/b]
Los Kekkei Tōta son combinaciones de tres elementos básicos. Para utilizarlos, el usuario debe dominar los tres elementos involucrados y gastar [color=#ff4136]dos slots elementales[/color] adicionales para habilitar su uso. Estos elementos son extremadamente raros y poderosos, y solo algunos personajes específicos pueden alcanzarlos.

El ejemplo más representativo de Kekkei Tōta es:

[color=#ffffff]Jinton (Elemento Polvo):[/color] combinación de Doton, Fuuton y Suiton.

Debido a su complejidad, los Kekkei Tōta no solo poseen propiedades especiales, sino que también interactúan de manera diferente en los choques elementales, especialmente al enfrentar Kekkei Genkai.

[hr]

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Energías Yin - Yang [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]

Además de los cinco elementos naturales, el chakra puede manifestarse en dos naturalezas adicionales: Yin y Yang. A diferencia de los elementos convencionales, estas naturalezas no están ligadas al entorno físico, sino que provienen directamente de la energía espiritual (Yin) y la energía física (Yang) del usuario. No requieren el uso de slots elementales, ya que no funcionan como elementos ofensivos o defensivos en sí mismos, sino como una guía conceptual que delimita los efectos posibles de un jutsu. Gracias a esta distinción, se evita que técnicas elementales adquieran propiedades ilógicas, como fuego que cure o rayos que controlen la mente.

[color=#FF4136]Yin:[/color]
El Yin representa el mundo espiritual, mental, la imaginación, los sentidos, la percepción y la ilusión. Abarca todo lo que manipula la mente, la conciencia o la espiritualidad, y se utiliza principalmente en técnicas de Genjutsu, ilusiones y manipulaciones del alma.

Se divide en tres grandes ramas:
[list]
[*][b]Manipulación de Conciencia: [/b]Técnicas que afectan directamente la mente del enemigo. Esto puede ir desde confundir al enemigo hasta controlarlo por completo. Dentro de esta categoría entran técnicas como el Shintenshin no Jutsu del clan Yamanaka, que permite trasladar la conciencia al cuerpo del enemigo para dominarlo.
[*][b]Manipulación de Cuerpo a través de la Mente:[/b] Aunque sigue siendo Yin, en este caso no se busca controlar la mente sino dominar el cuerpo del oponente desde afuera, tomando control de su cuerpo sin afectar su conciencia. Técnicas como el Kagemane no Jutsu del clan Nara entran aquí, ya que atan el cuerpo a través de la sombra, forzándolo a moverse según los gestos del usuario.
[*][b]Engaño Sensorial o Ilusorio:[/b] Aquí se encuentran los Genjutsus clásicos, aquellos que afectan la percepción sensorial del oponente, haciéndolo ver, oír o sentir cosas que no están ocurriendo. El Kawarimi no Jutsu, si bien es una técnica de escape, también entra como ejemplo de uso del Yin, ya que engaña momentáneamente los sentidos del rival haciéndole creer que su ataque fue efectivo.
[/list]

[color=#FF4136]Yang:[/color]
El Yang representa el mundo físico, la vitalidad, el cuerpo, la carne, la regeneración y el crecimiento. Son técnicas que afectan directamente la materia viva, el chakra físico o el cuerpo en sí, tanto propio como ajeno. Se utiliza principalmente en técnicas médicas, de modificación corporal o de potenciación física.

También se divide en tres ramas distintas:
[list]
[*][b]Curación y Vitalidad: [/b]Técnicas enfocadas en sanar heridas, restaurar tejidos, revitalizar el cuerpo o eliminar elementos nocivos del organismo como toxinas, venenos, quemaduras o infecciones. Ejemplo: el Saikan Chūshutsu no Jutsu, que permite eliminar venenos del cuerpo del objetivo utilizando agua sin necesidad de poseer el elemento Suiton.
[*][b]Alteración Física o Morfológica:[/b] Aquí entran todas las técnicas que modifican el cuerpo, ya sea para aumentar su tamaño, estirarlo, cambiar su forma o endurecerlo. Ejemplo: el Baika no Jutsu del clan Akimichi, que permite agrandar partes del cuerpo o al cuerpo completo para potenciar el daño físico.
[*][b]Manifestación Física de lo Abstracto:[/b] Técnicas que permiten darle forma tangible a cosas normalmente inmateriales, como la energía o el chakra. También entran aquí las técnicas que imbuyen el cuerpo con chakra de forma que este se fortalezca más allá de los límites normales. Un ejemplo es el chakra médico endurecido, capaz de convertir una mano normal en un bisturí de precisión o en una garra de destrucción.
[/list]
Aunque ambas naturalezas pueden combinarse para formar técnicas Yin-Yang, que son capaces de crear algo a partir de la nada, manipular la realidad, regenerar estructuras físicas y energéticas, o dar vida a creaciones,[color=#FF4136] las técnicas Yin-Yang no se encuentran disponibles para la creación de jutsus por parte de los usuarios.[/color]

[hr]

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Ventaja Elemental [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]

Cada elemento básico posee una ventaja específica sobre otro, siguiendo una cadena de dominancia elemental:
[list]
[*]Katon vence a Fuuton
[*]Fuuton vence a Raiton
[*]Raiton vence a Doton
[*]Doton vence a Suiton
[*]Suiton vence a Katon
[/list]
Los elementos avanzados, como los Kekkei Genkai (combinaciones de 2 elementos) y los Kekkei Tōta (combinaciones de 3), también aplican este sistema de ventajas.

Un Kekkei Genkai tiene ventaja sobre cualquier jutsu que utilice, de forma individual o combinada, los elementos básicos incluidos en su composición. Por ejemplo, un elemento Hielo (Suiton + Fuuton) tendrá ventaja contra jutsus de Suiton, Fuuton, o cualquier  otro Kekkei Genkai que los mezcle.

Los Kekkei Tōta, al ser fusiones de tres elementos, son superiores tanto a los elementos básicos como a los Kekkei Genkai, ya que tienen ventaja sobre cualquier técnica que incluya elementos débiles frente a los tres que lo componen, sean estos individuales o combinados. Por ejemplo, el elemento Polvo (Doton, Fuuton y Suiton) es fuerte contra técnicas con Suiton, Raiton o Katon, y todas sus posibles combinaciones. Sin embargo, solo puede ser contrarrestado por otro Kekkei Tōta que contenga exactamente Raiton, Katon y Doton, los tres elementos con ventaja directa sobre sus componentes.

[hr]

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Bonus por Ventaja Elemental [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]

Cuando un jutsu entra en choque directo contra otro y posee una ventaja elemental legítima, el jutsu con ventaja obtiene un bono adicional de daño. Este bono consiste en añadir el valor del Modificador de Control de Chakra (MCCK) multiplicado por 10 al daño total del jutsu [i][MCCKx10][/i].

Este bono no consume chakra adicional, ni requiere elevar el nivel del jutsu, ya que se considera una consecuencia natural de la ventaja elemental. Solo aplica en situaciones donde dos técnicas entran en conflicto directo, y no se aplica si el jutsu simplemente golpea a un enemigo sin que exista choque contra otra técnica. Además, este bono está calculado en el Jutsu Base del Árbol Elemental indicando además a que elementos puede aplicar dicho bono.

[size=xx-large][b]Clanes y Estilos[/b][/size][/align]

Los clanes son facciones que han desarrollado y perfeccionado sus jutsus a lo largo de generaciones, transmitiendo sus secretos desde la infancia y alcanzando un dominio único en sus técnicas.

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Tendencia Elemental [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]

Todos los clanes poseen una [color=#FF4136]tendencia elemental[/color], la cual puede estar compuesta por entre 1 y 3 elementos básicos o, en su defecto, Yin o Yang.

La tendencia elemental permite al usuario al momento de desbloquear un elemento, garantizar que [color=#FF4136]los primeros dados elementales sean asignados a dichos elementos[/color], y esto se repite hasta que el usuario complete [color=#FF4136]todos los elementos de su tendencia.[/color]

En el caso de los clanes con Kekkei Genkai, los usuarios podrán obtener ese elemento único; sin embargo, no podrán crear nuevos Kekkei Genkai a menos que posean [color=#FF4136]la capacidad para crear uno nuevo[/color].

[b]Ejemplo: [/b]Si un personaje pertenece al Clan Yuki, [color=#FF4136]podrá utilizar Hyōton desde nivel 1.[/color] Sin embargo, las ventajas elementales de este Kekkei Genkai solo estarán activas una vez [color=#FF4136]desbloquee Suiton y Fūton y pague el slot elemental de Hyōton[/color]. Hasta ese momento, sus jutsus podrán ser usados, pero no tendrán [color=#FF4136]ventaja elemental frente a otros elementos[/color].

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Estilos [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]

Además de los clanes, existe una división llamada Estilos. Los estilos se dividen en dos tipos:
[list]
[*][color=#FF4136]Estilos Totales: [/color]No poseen tendencia elemental.
[*][color=#FF4136]Estilos de Clanes: [/color]Poseen tendencia elemental de la misma forma que un clan.
[/list]
La mayor diferencia entre clanes y estilos es que los estilos pueden enseñarse a otras personas [color=#FF4136]usando el sistema de Maestros del Foro[/color]. No obstante, cuando un estilo es enseñado, [color=#FF4136]no se transfiere la habilidad base del árbol[/color]. En otras palabras, [color=#FF4136]aprender el Estilo Hyūga no otorga el Byakugan.[/color] En el caso de los clanes con estilo, si alguien roba su ADN,[color=#FF4136] no aprenderá automáticamente el estilo de sus jutsus.[/color]

Los estilos solo pueden obtenerse de dos maneras:
[list]
[*]Desde el inicio del personaje.
[*]Siendo enseñados por otro usuario.
[/list]

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Límite de Clanes y Estilos [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]

Un usuario solo puede manejar un máximo de [color=#FF4136]dos clanes y/o estilos simultáneamente[/color]. Puede tener dos clanes, o dos estilos, o bien un clan y un estilo. Esta regla aplica para [color=#FF4136]todo tipo de combinaciones[/color], si por ejemplo, un usuario obtiene dos clanes a través de una virtud, entonces ese usuario no podrá aprender un nuevo estilo, así mismo, si alguien posee ya dos estilos y adquiere un ADN, por mucho que se implante, por ejemplo, un Sharingan de manera exitosa, su mente no será capaz de procesar dicha información y será un simple ojo, un Sharingan simplemente estético pero no funcional.

- Concepto: los elementos son la base del estilo de combate; permiten jutsus elementales y combinaciones avanzadas. Incluye básicos + avanzados + Yin/Yang.
- Slots elementales: máximo 5 slots. Elemento básico = 1 slot; Kekkei Genkai = 1 slot extra; Kekkei Tōta = 2 slots extra.
- Elementos básicos: Katon, Suiton, Doton, Fuuton, Raiton. Se obtienen comprando árbol o por tirada aleatoria.
- Avanzados: Kekkei Genkai (2 elementos): requiere 2 elementos base + 1 slot extra (ej. Hyōton = Agua + Viento). Kekkei Tōta (3 elementos): requiere 3 elementos base + 2 slots extra; muy raros y superiores.
- Yin y Yang (NO usan slots): Yin (espiritual/mental): ilusiones, control mental, manipulación indirecta del cuerpo. Yang (físico/vital): curación, buffs físicos, transformación corporal.
- Ventaja elemental (ciclo): Katon → Fuuton → Raiton → Doton → Suiton → Katon.
- Reglas avanzadas: Kekkei Genkai tiene ventaja sobre sus elementos base; Kekkei Tōta es superior a básicos y genkai.
- Bonus por ventaja: solo en choque de jutsus; daño extra [MCCKx10]; NO cuesta chakra; NO sube el nivel del jutsu.
- Clanes: tienen afinidad/tendencia elemental, dan los primeros elementos y pueden dar acceso a Kekkei Genkai.
- Estilos: pueden aprenderse; NO incluyen el jutsu base del clan; máximo 2 en total (clanes + estilos combinados).
- Regla clave: más elementos ≠ mejor; la ventaja real está en cómo los combinas y cuándo los usas.
- Resumen: Elementos = poder; Combinación = maestría; Decisión = victoria.
