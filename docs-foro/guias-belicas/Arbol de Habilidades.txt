[b]¿Qué es un Árbol de Habilidad?[/b]

Un Árbol de Habilidad es una estructura de progresión personalizada que combina tres ramas especializadas y un jutsu base único. Este sistema permite que los usuarios desarrollen habilidades en profundidad, eligiendo una rama inicial y desbloqueando nuevas mejoras y especialidades a medida que invierten recursos en el árbol. El Árbol puede representar un estilo de combate específico (como Taijutsu, Ninjutsu, Bukijutsu o Genjutsu) tanto estilos ofensivos como defensivos o estar vinculado a una herencia particular (como los árboles de clanes).

Cada [color=#ff4136]rama [/color]dentro del árbol ofrece un [color=#ff4136]Jutsu Base de Rama, 6 mejoras y 3 especialidades[/color] que alteran la naturaleza del combate o las reglas del sistema. A medida que se desarrollan las ramas, el jutsu base se adapta y se vuelve más versátil o potente, reflejando el crecimiento del personaje dentro del árbol de habilidades.

[align=center][img]https://i.imgur.com/uLlSxrv.png[/img][/align]

A continuación, desglosaremos este esquema utilizando como ejemplo el jutsu insignia del Clan Nara, junto con su rama física, para ilustrar cada parte del árbol:

[b][color=#FF4136]Jutsu Base: [/color][/b]El jutsu base representa la habilidad esencial del árbol. Este jutsu puede ser activo o pasivo y define en gran parte el estilo del usuario. Cada árbol posee su propio jutsu base, que será único e irrepetible en otros árboles. Un ejemplo de Jutsu Base es el Kagemane no Jutsu que presentamos a continuación.
[tecnica=NARA001]

[color=#FF4136][b]Jutsu Base de Rama: [/b][/color]Cada rama incluye su propio jutsu base de rama. Este jutsu puede tener su propia funcionalidad o ser una mejora del jutsu base. Es una condición para que las mejoras de la rama funcionen. Un ejemplo de Jutsu Base de Rama seria la mejora del Kagemane no Jutsu, el Kage Butai, que funciona como una mejora del mismo Kagemane, funcionando en función del mismo y cambiando sus propiedades.
[tecnica=NARA301]

[color=#FF4136][b]Mejoras: [/b][/color]Las mejoras son efectos adicionales que se pueden aplicar al jutsu base de rama para aumentar su eficacia, impacto o utilidad. Las mejoras suelen estar etiquetadas por efectos concretos, como por ejemplo: DAÑO [MCCKx1], donde MCCK se refiere al "Modificador de Control de Chakra". Este formato permite a los jugadores identificar rápidamente las mejoras que se alinean con su estilo sin necesidad de leer descripciones extensas.
[tecnica=NARA305]

[color=#FF4136][b]Especialidades[/b][/color]
Las especialidades son beneficios de alto nivel que se obtienen conforme se desarrolla una rama. Estas habilidades permiten modificar profundamente el funcionamiento del jutsu base o del árbol en general, reduciendo costos, alterando la ejecución de técnicas, aumentando el límite de mejoras, habilitando nuevas formas de uso entre otros posibles y usos. Un ejemplo es el Kagetai no Jutsu de los Nara, que adelgaza la sombra a tal punto de que esta obtiene propiedades de Sigilo durante su ejecución.
[tecnica=NARA931]

[color=#FF4136][b]Chakra[/b][/color]
El recurso principal para ejecutar técnicas es el chakra. Cada jutsu posee un costo que varía según su nivel y los efectos que incorpora. A medida que se agregan mejoras, el consumo de chakra también aumenta, especialmente si estas aportan propiedades ofensivas, defensivas o pasivas de gran impacto. Por ello, es fundamental saber cuándo aplicar las mejoras y cuándo reservarlas, ya que un uso estratégico del chakra puede marcar la diferencia entre mantener el control del combate o agotar rápidamente los recursos del personaje.

[color=#FF4136][b]Niveles del Jutsu[/b][/color]
El nivel total de un jutsu se determina sumando el nivel del jutsu base junto con el nivel de todas las mejoras aplicadas. El valor máximo permitido es 5. Este límite obliga al usuario a priorizar entre daño, utilidad, y propiedades ofensivas o defensivas, favoreciendo la toma de decisiones estratégicas durante el combate.

No existe un nivel mínimo requerido para ejecutar un jutsu, pero el tope de nivel fue establecido para garantizar un balance justo entre todas las técnicas, evitando que un jutsu supere ampliamente a otro sin justificación. Este límite se aplica por igual sin importar si el jutsu proviene de un árbol dependiente, independiente o integrado: el nivel máximo siempre será 5.

En caso de que el jutsu base permita incorporar mejoras de múltiples ramas de forma simultánea, estas también deben respetar el límite total de nivel. Cabe destacar que esta restricción se aplica por jutsu ejecutado. Si un personaje realiza dos jutsus distintos al mismo tiempo, cada uno contará con su propio límite de nivel, siendo 5 el máximo para cada uno por separado.

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Compras de Ramas y Niveles de Ramas [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]

Las ramas y sus niveles se desbloquean utilizando Tobis, la moneda o recurso principal para personalizar un Árbol de Habilidades.

Al obtener un nuevo Árbol de Habilidades, el usuario comenzará únicamente con el jutsu base del árbol y una rama activa, la cual incluirá su propio jutsu base de rama. Las demás ramas deberán desbloquearse pagando su costo correspondiente, y al hacerlo, solo se entregará el jutsu base de esa rama.

Cada vez que se sube el nivel de una rama, el usuario podrá elegir dos de las seis mejoras disponibles para esa rama. Cada rama puede mejorar hasta 3 niveles en total.

A medida que se acumulan mejoras dentro del árbol, se habilita la selección de Especialidades como recompensa por alcanzar ciertos hitos. Por cada tres niveles acumulados [color=#ff4136](ya sea subiendo una misma rama o distribuyéndolos entre distintas ramas)[/color], se desbloqueará una Especialidad. [color=#ff4136]El máximo de especialidades posibles por árbol es 3[/color], alcanzables solo al subir[b] los 9 niveles totales disponibles en cada Árbol de Habilidades.[/b]

El costo en Tobis de cada árbol y de cada mejora dentro de una rama está detallado en la página correspondiente. Estos precios escalan progresivamente: mientras más compras realiza el usuario dentro de un mismo árbol, mayor será el costo de futuras ramas y mejoras.

[align=center][img=25x25]https://i.imgur.com/r8SokMx.png[/img][color=#e82a1f][size=xx-large] Limite de Árboles de Habilidad [/size][/color][img=25x25]https://i.imgur.com/r8SokMx.png[/img][/align]

El sistema de niveles comprende desde el Nivel 1 hasta el Nivel 20, y el progreso de cada ninja está marcado por la cantidad de Árboles de Habilidad que puede tener como máximo por nivel.

Cada Árbol de Habilidad representa un aspecto importante del desarrollo del ninja, como el dominio de un elemento, una especialidad de combate, o una técnica avanzada.
[list]
[*]Nivel 1: 1 árbol
[*]Nivel 5: 2 árboles
[*]Nivel 10: 3 árboles
[*]Nivel 15: 4 árboles
[*]Nivel 20: 5 árboles
[/list]
Estos árboles desbloqueados no contemplan los árboles gratuitos que el usuario obtiene desde el momento que crea su ficha. O sea, no contempla ninguno de las tres ramas aprendidas en la academia ninja: [color=#FF4136]Taijutsu, Defensa, Bukijutsu y Resistencia[/color]. Tampoco contempla la adquisición del [color=#FF4136]árbol de Clan[/color] que se entrega por linaje.

Estos árboles que el usuario puede desbloquear son para usarse tanto en los árboles elementales como en las técnicas libres que el usuario pueda aprender. Además, también son los espacios donde el usuario podrá colocar sus árboles de habilidades únicas que haya creado o adquirido.

Imagen 1 — https://i.imgur.com/3sMo3uu.png  "ÁRBOLES DE HABILIDAD · resumen ultra corto"
- Concepto: 1 Árbol = 1 Jutsu Base + 3 Ramas. Cada rama tiene: 1 jutsu base de rama, 6 mejoras y 3 especialidades. Un árbol representa un estilo de combate o una herencia.
- Estructura: Jutsu Base (núcleo, define el estilo) → Jutsu Base de Rama (habilita la rama y sus mejoras) → Mejoras (efectos directos: daño, rango, etc.; se combinan para potenciar jutsus) → Especialidades (modificadores avanzados; cambian reglas, costos y funcionamiento).
- Nivel de jutsu: nivel = base + mejoras; MÁXIMO 5 por jutsu (siempre, sin excepción). Obliga a elegir entre Daño / Utilidad / Efectos.
- Progresión de ramas: empiezas con jutsu base + 1 rama activa; otras ramas se compran con Tobis; cada rama sube máx 3 niveles, y cada nivel deja elegir 2 mejoras.
- Especialidades: se desbloquean cada 3 niveles totales; máximo 3; requieren 9 niveles totales del árbol.
- Chakra: cada mejora sube el costo; la mala gestión te deja sin recursos.
- Límite de árboles por nivel: Nv1=1, Nv5=2, Nv10=3, Nv15=4, Nv20=5. NO incluye: Academia (Taijutsu, Defensa, Bukijutsu) ni el árbol de Clan.
- Regla clave: ningún jutsu puede superar el nivel 5 sin importar cuántas mejoras tengas.

Imagen 2 — https://i.imgur.com/XP9DErr.png  (tabla de COSTOS en Tobis)
- Desbloqueo de Árbol: 15 Tobis base, +10 por cada árbol ya desbloqueado (Árbol 1=15T, 2=25T, 3=35T...).
- Desbloqueo de Nueva Rama: 10 Tobis base, +5 por cada rama desbloqueada en cualquier árbol (Rama 1=10T, 2=15T, 3=20T...).
- Subida de Nivel de Rama (1 nivel): 5 Tobis base, +2 por cada nivel ya desbloqueado entre todas las ramas (Nivel 1=5T, 2=7T, 3=9T...).
